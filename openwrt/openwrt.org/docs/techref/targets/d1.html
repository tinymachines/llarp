d1

Support for the Allwinner D1 RISC-V based SoCs.

  • RISC-V single-core T-Head C906 (RV64GCV)
  • Tensilica HiFi4 DSP
  • DDR2/DDR3 support
  • 10/100/1000M ethernet
  • usual peripherals like USB2, SPI, I2C, PWM, etc.
