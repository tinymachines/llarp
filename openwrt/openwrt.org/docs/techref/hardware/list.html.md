# OpenWrt Hardware List

This page tries to document various companies involved in distributing hardware that can be used with OpenWrt.

![:!:](/lib/images/smileys/exclaim.svg) There is a difference between advertising Open Source support, OpenWrt support and continuous involvement in OpenWrt development processes.

A database of supported hardware is available in this wiki: [Table of Hardware](/toh/start "toh:start")

Another detailed database with detailed Chip, FCC and Vendor (ODM, brand) annotation is maintained at [Wikidevi](http://wikidevi.com/ "http://wikidevi.com/") project.

## SoC manufacturer and HW vendor

For a list of SoC see: [SoC (System on a Chip)](/docs/techref/hardware/soc "docs:techref:hardware:soc")

Some brands use many different ODM/OEM companies and might change SoC between revisions: extreme example is [D-Link DIR-615](https://wikidevi.com/wiki/D-Link_DIR-615 "https://wikidevi.com/wiki/D-Link_DIR-615").

TP-Link seems to prefer QCA SoCs. Asus seems to prefer Broadcom SoCs.

### Additional Information

Arris sells Cable modem/routers. Similar to DSL chipsets most cable (DOCSIS) chipsets are likely closed source, making OpenWrt support unlikely.
