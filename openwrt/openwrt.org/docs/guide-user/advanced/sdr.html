What can you do with SDR?

With SDR you can receive

  • 433 MHz / 868 MHz weather sensors (temperature, humidity, a few sensors send also pressure)
  • AIS
  • ACARS
  • POCSAG
  • LTE
  • Amateur radio
  • Radio broadcasting
