Smartmontools

This article refers to Smartmontools. You can install it by

Installation

opkg update
opkg install smartmontools

Update Drive Database (drivedb.h)

To save flash space there is no drivedb included in the OpenWRT package of smartmontools. The automatic update of drive.db described here is missing as well. In case you can affort/waste ~215kB (12-April-2020) you can download the current database from GIT Drivedb.h. Transfer it via SCP or WinSCP and place it at 

mkdir /usr/share/smartmontools
mv drivedb.h /usr/share/smartmontools/drivedb.h
