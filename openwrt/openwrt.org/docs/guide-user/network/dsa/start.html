DSA Networking

Starting in OpenWrt 21.02, many target architectures have switched from swconfig to DSA (Distributed Switch Architecture), the Linux kernel subsystem for network switches. DSA configuration is incompatible with swconfig.

See the links below to learn more about using OpenWrt and DSA.

  • DSA Mini-Tutorial gives an overview of the concepts of DSA and configuration examples
  • Converting to DSA tells how to convert your swconfig configuration files to DSA
  • Upgrading to OpenWrt 21.02.0 gives other recommendations and requirements for upgrading to OpenWrt 21.02
