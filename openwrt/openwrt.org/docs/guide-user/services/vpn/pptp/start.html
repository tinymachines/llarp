PPTP

PPTP is considered insecure due to multiple vulnerabilities. Use another VPN solution if possible.

The PPTP is a method for implementing VPN. PPTP uses a control channel over TCP and a GRE tunnel operating to encapsulate PPP packets. pptpd is server daemon which enables PPTP clients to establish tunnel over IP network.
