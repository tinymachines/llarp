LuCI themes

The default LuCI theme is called Bootstrap. Install other themes via Packages, go to LuCI → System → Software, filter for “luci-theme”, you will see several available.

To change themes go to System → System → Language and Style. Use the Design dropdown to select the theme, then click save and apply:

Bootstrap theme

Bootstrap is the default theme, it has dark mode and light mode options.

Material theme

Material is inspired by the Google Material Design.

OpenWrt2020 theme

OpenWrt2020 is an old default theme and still available.

OpenWrt theme

OpenWrt is an old default theme and still available.

Other themes

  • Argon is a popular theme with light and dark modes available on a separate repository, found on the Argon main thread.
  • Neobird is a clean, beautiful theme with mobile support.
  • If you wish to create your own themes see HowTo: Create Themes.
