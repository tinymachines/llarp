Update wikidata

  1. Login to https://www.wikidata.org/
  2. Go to https://www.wikidata.org/w/index.php?title=Q1140401&uselang=en
  3. Scroll down to the end of the “software version identifier” section
  4. Click “Add value”
  5. Enter release number
  6. Click on the leftmost icon to the left of the input box and change from “Normal rank” to “Preferred rank”
  7. Click “Add qualifier”
  8. Type “version type”, click on the proposed item, then add “stable release”
  9. Click “Add qualifier”
  10. Type “publication date”, click on the proposed item, then add the publication date as stated in the release notes
  11. Click “Add reference”
  12. Enter url to release notes in the wiki
  13. Click “Publish”
