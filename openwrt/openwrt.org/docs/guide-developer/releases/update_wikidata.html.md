# Update wikidata

01. Login to [https://www.wikidata.org/](https://www.wikidata.org/ "https://www.wikidata.org/")
02. Go to [https://www.wikidata.org/w/index.php?title=Q1140401&amp;uselang=en](https://www.wikidata.org/w/index.php?title=Q1140401&uselang=en "https://www.wikidata.org/w/index.php?title=Q1140401&uselang=en")
03. Scroll down to the end of the “software version identifier” section
04. Click “Add value”
05. Enter release number
06. Click on the leftmost icon to the left of the input box and change from “Normal rank” to “Preferred rank”
07. Click “Add qualifier”
08. Type “version type”, click on the proposed item, then add “stable release”
09. Click “Add qualifier”
10. Type “publication date”, click on the proposed item, then add the publication date as stated in the release notes
11. Click “Add reference”
12. Enter url to release notes in the wiki
13. Click “Publish”
